# Exploratory / Not Required for Paper Reproduction

If you landed in this folder directly: **you don't need anything here to
reproduce `Paper5_FINAL.md`.** See `../SCRIPT_INDEX.md` at the repo root for
the full breakdown, or `../REPRODUCTION_GUIDE.md` for exact reproduction
commands.

Two things live here:

- **`ml_cohort_lineage_v12_to_v33/`** — the 22 intermediate script versions
  between the leakage-bug demo (v11) and the final canonical result (v34).
  Debugging history, not individually cited in the paper.
- **`live_deployment_stack/`** — exploratory scripts toward a real-time
  wearable deployment, separate from this paper's software-verification
  scope. Not cited in the paper.

Kept in the repo for transparency rather than deleted, in keeping with this
project's disclosure-over-omission approach to the rest of the pipeline.
